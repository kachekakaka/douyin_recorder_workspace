# FFmpeg distribution notice

The Windows package includes an unmodified FFmpeg 7.1 full build published by
Gyan Doshi. The exact release asset and SHA-256 are recorded in
`packaging/release-lock.json`.

The selected build is distributed under GPL-3.0-or-later. The release builder
copies every license and notice file shipped in the selected archive into
`licenses/ffmpeg/` and includes the build-provider notice separately.

Corresponding source and build information:

- https://github.com/FFmpeg/FFmpeg/tree/n7.1
- https://ffmpeg.org/
- https://www.gyan.dev/ffmpeg/builds/

No local modification to FFmpeg is made by this project.

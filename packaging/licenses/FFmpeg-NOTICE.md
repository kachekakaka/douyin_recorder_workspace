# FFmpeg distribution notice

The Windows package includes an unmodified LGPL shared build produced by
`BtbN/FFmpeg-Builds` from the fixed release tag recorded in
`packaging/release-lock.json`.

FFmpeg is primarily licensed under LGPL-2.1-or-later. The release builder copies
all license and notice files shipped in the selected FFmpeg archive into
`licenses/ffmpeg/`. The BtbN build scripts are MIT licensed; their notice is
included separately.

Source and build instructions:

- https://ffmpeg.org/
- https://github.com/FFmpeg/FFmpeg
- https://github.com/BtbN/FFmpeg-Builds

No local modification to FFmpeg is made by this project.

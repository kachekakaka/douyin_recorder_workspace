# 11. 安全与凭据边界

- 房间 URL 由抖音解析器校验 host、scheme、重定向目标，防止 SSRF。

- FFmpeg 只使用 Python 构造的 argv；用户输入不得直接成为任意命令行参数或输出路径。

- 媒体和导出 API 只接收数据库对象 ID，不接收浏览器提供的服务器绝对路径。

- 目录使用 room_key/session UUID，不使用昵称作为唯一目录名；所有路径必须校验位于 records/ 下。

- Cookie、sessionid、签名、管理员密码和初始化令牌不进入 Git、日志、数据库普通字段或 API。

- 监听非回环地址时强制管理员认证、CSRF、可信 Host、登录限速、安全 Cookie 和审计日志。

- 同源部署；默认 CSP、X-Frame-Options、nosniff、Referrer-Policy；API 响应 no-store。

- 原始事件按最小必要原则保存；设置保留期限和可审计删除任务。

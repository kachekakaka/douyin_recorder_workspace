## Security boundaries

All v0.1.1 public reports exclude private configuration values, absolute runtime paths, Cookie,
token, complete signed URLs, raw payload and raw frame. Runtime backups may contain private
configuration and must remain outside Git.

Database maintenance never deletes business rows or media. Phase 2 does not execute VACUUM.
Write maintenance fails closed unless a verified backup exists and an exclusive database lock can
be acquired.

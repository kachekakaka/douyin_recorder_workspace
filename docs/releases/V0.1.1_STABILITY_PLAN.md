# v0.1.1 Stability Foundation Plan

## 定位

v0.1.1 是 v0.1.0 Release 后的工程稳定性版本。

本版本不修改真实 recipient 协议判断，不改变 `live_verified=false`，不关闭 Issue #1。

## 目标

提升单机 Windows 长时间运行、故障恢复、诊断和维护能力。

## 范围

### 1. Runtime diagnostics

- 增加统一诊断报告生成入口；
- 收集版本、配置摘要、数据库 schema、最近错误类型；
- 禁止输出 Cookie、完整 URL、签名参数、raw payload。

### 2. Recovery

- 增加 SQLite integrity check 工具；
- 增加 backup restore 验证；
- 增加异常退出恢复测试。

### 3. Maintenance

- 增加运行数据清理策略文档；
- 增加数据库维护命令；
- 增加长期运行 smoke 测试。

## 明确不做

- 不接入真实 recipient 协议；
- 不修改 protobuf contract；
- 不增加 Redis/Celery/PostgreSQL；
- 不增加多机架构；
- 不增加 OCR、人脸、声纹、礼物、弹幕 fallback。

## 验收

每个 PR 必须：

- feature branch；
- commit；
- push；
- PR；
- CI green；
- merge main。

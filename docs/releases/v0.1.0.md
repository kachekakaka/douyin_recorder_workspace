# Douyin Recorder v0.1.0

首个可恢复的 Windows x64 工程发布版本。

## 已包含

- 单进程多房间直播状态轮询与自动录制；
- 单房间 Recording Session、FFmpeg 分片和 SQLite 审计；
- Waiting / Active / Unknown recipient 时间线投影；
- 持久化 postprocess jobs、retry/cancel/recovery 和区间导出；
- Windows 便携 Python 3.13 与 Gyan FFmpeg 7.1 full build；
- manifest、依赖许可证、SHA-256、源码 ZIP 和 Git Bundle。

FFmpeg 使用固定长期 Release 资产：

```text
https://github.com/GyanD/codexffmpeg/releases/download/7.1/ffmpeg-7.1-full_build.zip
SHA-256: FA074A46B7BB862E37AA27CCD0522D22A1FE90AEC3A950C8E0C0C8622E0A03D1
```

该 URL 与摘要同时由 Microsoft winget 官方清单固定；包内包含 GPL-3.0-or-later 许可证和对应源码/构建来源说明。

## 发布验证

- Windows 包必须在新的干净目录解压；
- 便携版 `verify.bat`、loopback `/api/status` 与三类 FFmpeg smoke 必须通过；
- ZIP 成员按 POSIX 相对路径字符串排序，Linux 与 Windows 生成顺序一致；
- source ZIP 与 Git Bundle 必须恢复到和发布提交一致的 Git tree；
- Release 资产必须通过各自的 SHA-256 清单复核。

## 重要限制

- 仅允许本机 loopback 使用，尚无管理员认证和公网管理；
- `WebcastGroupLiveGiftRecipientRecommendMessage` 的真实现场字段仍未通过人工审查 fixture 完成验证；
- `live_verified=false`，Issue #1 继续开放；
- 不提供账号登录、验证码/风控绕过或互动消息发送。

P2A 范围

  • 房间 CRUD、严格 URL 边界和公开直播页立即检查
  • FLV/HLS/清晰度候选的内存解析与脱敏结果
  • 显式 start/stop、单房间唯一 active Session、progress 与 segment 持久化
  • 每个 enabled room 一个串行 worker，全局限制并发检查
  • live 自动确保录制；连续 offline 达阈值停止；unknown/blocked/error 保持媒体
  • recording 与 recipient Waiting/Active/Unknown 共用同一个 session_id
  • 不包含真实 recipient 自动接入、直播后导出任务、多机器或公网管理

目标消息仍未形成经审查现场 fixture；运行 contract 保持 live_verified=false。
